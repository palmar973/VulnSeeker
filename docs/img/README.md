# Recursos visuales del README

Deja aquí las capturas que el `README.md` y el `README.en.md` referencian. Al usar
exactamente estos nombres de archivo, los marcadores de imagen se rellenan solos:

| Archivo | Contenido esperado | Formato sugerido |
|---------|--------------------|------------------|
| `vulnseeker-scan.gif` | GIF corto de un escaneo en la terminal (fase por fase) | GIF, ≤ 5 MB, ~1200 px de ancho |
| `vulnseeker-gui.png`  | Captura de la interfaz gráfica (CustomTkinter) | PNG, ~1600 px de ancho |
| `vulnseeker-report.png` | Captura de una página del reporte PDF generado | PNG, ~1200 px de ancho |

Consejos:

- Para el GIF puedes grabar con [ScreenToGif](https://www.screentogif.com/) (Windows) y recortar a 10-15 s.
- Comprime los PNG (por ejemplo con [TinyPNG](https://tinypng.com/)) para que el README cargue rápido.
- Mantén los nombres tal cual; si cambias uno, actualiza también el marcador en ambos README.
