## рҹ”Қ ВҝQuГ© es VulnSeeker?

**VulnSeeker Enterprise** es un escГЎner de vulnerabilidades web avanzado, desarrollado en Python, diseГұado para realizar auditorГӯas de seguridad automatizadas de forma integral. A diferencia de escГЎneres bГЎsicos que se limitan a un solo vector de ataque, VulnSeeker orquesta **16 mГіdulos independientes** de anГЎlisis en paralelo вҖ” desde inyecciones SQL y XSS hasta reconocimiento OSINT de subdominios y anГЎlisis con Inteligencia Artificial.

El sistema opera bajo una arquitectura modular extensible donde cada mГіdulo es un componente independiente que se registra en el motor central, permitiendo agregar nuevas capacidades de escaneo sin modificar el nГәcleo del sistema.

### ВҝPor quГ© VulnSeeker?

- **Cobertura real.** No es un script que lanza payloads a ciegas. VulnSeeker ejecuta una cadena completa: reconocimiento вҶ’ fingerprinting вҶ’ crawling вҶ’ ataque multihilo вҶ’ anГЎlisis IA вҶ’ reporte.
- **Inteligencia Artificial integrada.** Llama 3.3 70B actГәa como un CISO virtual que traduce hallazgos tГ©cnicos en riesgos de negocio ejecutivos.
- **Persistencia histГіrica.** Cada escaneo se almacena en SQLite, permitiendo anГЎlisis de tendencias y comparaciГіn entre auditorГӯas.

## вҡЎ CaracterГӯsticas Principales

### MГіdulos de Ataque

| MГіdulo | DescripciГіn | Severidad |
|--------|-------------|-----------|
| **SQL Injection** | DetecciГіn de SQLi basada en errores (MySQL, PostgreSQL, Oracle, MSSQL) | рҹ”ҙ HIGH |
| **Cross-Site Scripting** | IdentificaciГіn de XSS reflejado mediante inyecciГіn de canarios | рҹҹ  MEDIUM |
| **Command Injection** | DetecciГіn de ejecuciГіn remota de comandos (RCE) en parГЎmetros | рҹ”ҙ CRITICAL |
| **Local File Inclusion** | Pruebas de traversal de directorios para lectura de archivos locales | рҹ”ҙ HIGH |
| **Remote File Inclusion** | DetecciГіn de inclusiГіn de archivos remotos en aplicaciones web | рҹ”ҙ HIGH |

### MГіdulos de Reconocimiento y AnГЎlisis

| MГіdulo | DescripciГіn |
|--------|-------------|
| **Web Crawler** | Explorador estructural que mapea URLs, formularios y puntos de entrada |
| **Tech Fingerprinter** | IdentificaciГіn de servidor, lenguaje backend y CMS/Framework |
| **Subdomain Scanner** | Descubrimiento pasivo OSINT vГӯa crt.sh y HackerTarget con validaciГіn de subdominios live |
| **Port Scanner** | Escaneo de puertos abiertos en el objetivo |
| **Path Fuzzer** | Descubrimiento de rutas y archivos ocultos mediante diccionarios |

### MГіdulos de ConfiguraciГіn y Hardening

| MГіdulo | DescripciГіn |
|--------|-------------|
| **Header Analyzer** | EvaluaciГіn de cabeceras de seguridad (CSP, HSTS, X-Frame-Options, etc.) |
| **Cookie Scanner** | AuditorГӯa de atributos de cookies (Secure, HttpOnly, SameSite) |
| **WAF Detector** | IdentificaciГіn de Web Application Firewalls activos |
| **CMS Auditor** | AnГЎlisis especГӯfico para WordPress, Joomla, Drupal y otros CMS |
| **Exposure Scanner** | DetecciГіn de archivos sensibles expuestos (.env, .git, backups, configs) |

### Inteligencia Artificial

| CaracterГӯstica | Detalle |
|---------------|---------|
| **Modelo** | Llama 3.3 70B vГӯa Groq API |
| **FunciГіn** | GeneraciГіn de informes ejecutivos CISO con anГЎlisis de riesgo de negocio |
| **Output** | Nivel de riesgo global, top amenazas, plan de acciГіn inmediato (48h) y recomendaciones estratГ©gicas |

### Reportes

- рҹ“„ **PDF profesional** con portada, grГЎficos de severidad (pie + barras), tabla de hallazgos y resumen IA
- рҹ“Ҡ **JSON** estructurado para integraciГіn con otras herramientas
- рҹ“Ӣ **CSV** compatible con Excel para anГЎlisis manual
- рҹ—„пёҸ **SQLite** con historial completo de escaneos, exportable y consultable

## рҹҡҖ InstalaciГіn

### Requisitos Previos
- Python 3.10+ 
- pip (gestor de paquetes)
- *(Opcional)* API Key de [Groq](https://console.groq.com/) para el mГіdulo de IA

### Pasos

```bash
# 1. Clonar el repositorio
git clone https://github.com/palmar973/VulnSeeker.git
cd VulnSeeker

# 2. Crear entorno virtual
python -m venv .venv

# 3. Activar entorno virtual
# Windows:
.venv\Scripts\Activate.ps1
# Linux/macOS:
source .venv/bin/activate

# 4. Instalar dependencias
pip install -r requirements.txt

# 5. Ejecutar la GUI
python gui.py
```

### Uso por CLI


## рҹҢҗ Sobre vulnseeker.software

El sitio web [vulnseeker.software](https://vulnseeker.software) es **exclusivamente una landing page** de presentaciГіn y documentaciГіn del proyecto. VulnSeeker es una **aplicaciГіn de escritorio local** вҖ” no existe ni existirГЎ una versiГіn ejecutable desde el navegador.

Esta decisiГіn es deliberada: un escГЎner de vulnerabilidades requiere acceso directo a la red, ejecuciГіn multihilo sin restricciones del navegador y la capacidad de realizar cientos de peticiones HTTP concurrentes con control total sobre headers, timeouts y payloads. La arquitectura local garantiza **mГЎxima potencia de escaneo** y **total control del operador** sobre la herramienta.
